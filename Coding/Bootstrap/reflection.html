Project Reflection

1. Which component did you find most flexible?

The card component was the most flexible. It allowed me to combine images, text, and layout in a clean and responsive way without needing custom CSS.

2. What challenges did you face with layout or alignment?

Getting elements to align perfectly inside the grid system took some trial and error. Especially balancing image sizes and text side by side in the About section.

3. If you could improve this with custom styles or JS, what would you add?

I would add animations with JavaScript for the modal and buttons, and apply custom fonts and gradients using CSS to make it look more personalized and unique.
